There are 17 variables in the taxi data set and 24.1 million observations.

  • PULocation ID and DOLocation ID is a numerical key which refer to pick up location and drop off location when we combine it with location dataset.

  • Passenger_count refers to the number of customers in that specific ride.

  • Tpep_pickup_datetime is the time which ride starts.

  • Tip_amount is the tip that was given from the customer.

  • Trip_distance refers to the distance of the ride

# Based on the distribution of trip distances and the fact that it would take about 30 miles to drive across the entirety of New York City,\
# we decided to use 30 as the criterion to classify trips as short or long distancce

sql_short = ('''SELECT count(*) AS ST_Count, avg(tip_amount) AS ST_Tips
                FROM taxi_2020
                WHERE trip_distance <= 30''')

cur.execute(sql_short)
data_short = cur.fetchone()
data_short = pd.DataFrame(data_short).transpose()

sql_long = ('''SELECT count(*) AS LT_Count, avg(tip_amount) AS LT_Tips
                FROM taxi_2020
                WHERE trip_distance > 30''')

cur.execute(sql_long)
data_long = cur.fetchone()
data_long = pd.DataFrame(data_long).transpose()
